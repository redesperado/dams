# dams

**相关技术：**  
__后端：__  
后端框架：SpringBoot  
数据库：MySQL  
ORM：MyBatis  
安全框架：shiro  
密码加密：MD5  
__前端:__  
前端框架：BootStrap + Layui  
__中间技术：__  
Node.js
